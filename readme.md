# Dashboard template with Antd and React

## Tech Stack

- ReactJs
- Antd Component Library
- react-router-dom for client side routing
- tanstack-query because it is amazing for server side state
- jotai for atomic state management
- mock service worker(msw) to develop frontend features in isolation with having to touch backend
- turbo for monorepo with react frontend and nestjs backend (just for setup)
- vite for its simplicity and speed
